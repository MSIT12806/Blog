# 建立第一篇文章

這篇文章示範目前 Blog MVP 的基本能力。

## 為什麼用 VitePress

VitePress 很適合用來建立以內容為核心的網站，尤其是文件站、技術部落格、知識庫。

它的優點包括：

1. Markdown 直接轉成靜態頁面。
2. 開發體驗簡單，啟動速度快。
3. 預設 SEO、導覽與主題能力已經很完整。

## 這個專案現在可以做什麼

1. 在本機預覽文章。
2. 編譯成靜態網站。
3. 透過 GitHub Actions 部署到 GitHub Pages。

## 下一步建議

1. 把你的文章逐篇搬到 `src/posts/`。
2. 增加首頁精選文章區塊。
3. 加入 RSS 與 sitemap。
